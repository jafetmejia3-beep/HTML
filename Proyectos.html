Este proyecto es un robot programado. El robot puede moverse, detectar obstáculos y realizar tareas simples como recoger objetos. Tiene una interfaz intuitiva y fácil de usar para controlarlo.
Proyecto IA
Este proyecto es una aplicación de inteligencia artificial desarrollada con Python. La aplicación puede reconocer rostros, analizar emociones y proporcionar recomendaciones personalizadas basadas en los datos del usuario. Tiene una interfaz intuitiva y fácil de usar para interactuar con el sistema.
